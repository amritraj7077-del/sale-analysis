# sale-analysis
# 📊 Sales Analysis using Python

A complete data analysis project that explores 12 months of sales data to uncover business insights using **Python**, **Pandas**, and **Matplotlib**.

---

## 🚀 Project Overview

This project analyzes one year of sales data from an electronics store. The dataset is cleaned, merged, and analyzed to answer important business questions such as:

- Which month generated the highest sales?
- Which city had the highest revenue?
- What time should advertisements be displayed for maximum purchases?
- Which products are most frequently sold together?
- Which products are the top-selling items?

The project demonstrates the complete data analysis workflow, from data cleaning to visualization and business insights.

## 📌 Key Insights

- Certain months generate significantly higher revenue.
- A few cities contribute the majority of total sales.
- Customer purchases peak during specific hours of the day.
- Some products are frequently purchased together, enabling bundle recommendations.
- High-selling products are not always the most expensive.
